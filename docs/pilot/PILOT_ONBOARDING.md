# EatClub Pilot Onboarding

## Prerequisites
- Android Phone (Android 8.0+)
- USB Cable

## Installation (Sideloading)
1. Enable "Install from Unknown Sources" in Developer Options.
2. Transfer `app-debug.apk` to phone.
3. Tap APK to install.

## First Run
1. Open App. 
2. Verify "EatClub" title screen.
3. Grant permissions (Storage/Camera if requested).

## Initial Inventory Setup
1. Go to "Manual Entry".
2. Add your top 5 kitchen items (e.g., Milk, Eggs, Rice).
3. Verify they appear in Inventory List.

## Daily Workflow
1. When buying: Use Draft/Manual entry.
2. When cooking: Check Suggestions.
3. When eating: Use "Consume" action (Future Phase) or manually adjust quantity.

## Reporting Issues
- Use `TRUTH_FAILURE_TEMPLATE.md` for logic errors.
- Screenshot crashes.
